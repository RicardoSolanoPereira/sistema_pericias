# Sistema de Controle de Perícias

Sistema local desenvolvido em Python para gestão de:

- Processos judiciais
- Prazos
- Andamentos
- Agendamentos
- Controle financeiro por processo

## Stack

- Python
- Streamlit
- SQLAlchemy
- SQLite
- Git

## Estrutura

Arquitetura modular com separação de:

- UI (Streamlit)
- Services (regras de negócio)
- Models (SQLAlchemy ORM)
- Banco de dados local
